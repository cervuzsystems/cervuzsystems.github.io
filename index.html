Main Features

  • Encryption with AES-256

    • Cervuz utilizes AES-256 encryption for every file in the archive, providing a reliable high level of security to protect your sensitive data.

  • Use of Unique Salt

    • Each file is encrypted with a unique salt, which is randomly generated, reducing the risk of brute-force attacks and increasing encryption security.

  • Data Compression

    • Archives are encrypted and compressed using CervuzEngine, saving storage space and making file management easier.

  • Good Error Handling

    • Cervuz has a robust error handling mechanism, including special handling for password errors and temporary file management to ensure process integrity.

  • Temporary File Deletion

    • Once the encryption or decryption process is complete, all temporary files used will be deleted automatically to keep the system clean and secure.
